# 👨‍💼 Employee Management System

A modern **Employee Management System** built using **ASP.NET Core MVC**, **Entity Framework Core**, **SQL Server**, **ASP.NET Identity**, and **AdminLTE**. This application provides secure authentication, employee management, and a professional dashboard for managing employee records efficiently.

---

## 📌 Project Overview

The Employee Management System is designed to simplify employee record management through a secure and user-friendly web application. It allows authenticated users to manage employee information, view dashboard statistics, and perform complete CRUD (Create, Read, Update, Delete) operations.

---

## ✨ Features

* 🔐 Secure Login & Registration using ASP.NET Identity
* 🛡 Authentication & Authorization
* 📊 Dashboard with Live Statistics
* 👨‍💼 Employee Management (CRUD)
* 📋 Employee Details Page
* 📈 Recent Employees Table
* 🎨 Responsive AdminLTE Dashboard
* 💾 SQL Server Database Integration
* ⚡ Entity Framework Core
